# Movie Recommendation System (Streamlit)

A movie recommendation web application built using Streamlit and Python.  
The system allows users to search for movies using natural language input and receive genre-based recommendations derived from a structured dataset.

## Features

- User account management (create, select, delete users)
- Natural language input for movie preferences
- Automatic genre extraction using keyword matching
- Personalized recommendations based on user history
- SQLite database for persistent storage
- Lightweight web interface built with Streamlit
- Uses a public movie dataset (no API key required)

## Install Dependencies
pip install streamlit requests

## How to Run Code
python3 -m streamlit run GUI.py
